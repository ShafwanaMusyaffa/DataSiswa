## 💻 Install

1. **Clone Repository**
```bash
git clone https://github.com/ShafwanaMusyaffa/dasis.git
cd dasis
composer install
npm install
copy .env.example .env
```

2. **Buka ```.env``` lalu ubah baris berikut sesuai dengan databasemu yang ingin dipakai**
```
DB_PORT=3306
DB_DATABASE=datasiswa
DB_USERNAME=root
DB_PASSWORD=
```

3. **Instalasi website**
```bash
php artisan key:generate
php artisan migrate
```

4. **Jalankan website**
```bash
php artisan serve
```
